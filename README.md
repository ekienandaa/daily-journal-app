# 📔 Daily Journal App

A personal productivity journaling web application built with React, featuring an Indonesian-language user interface designed for daily-use simplicity.

## ✨ Features

- **Daily entries** — Create, view, and manage journal entries organized by date
- **Indonesian-language UI** — Localized interface for native Indonesian speakers
- **Persistent storage** — Entries saved across sessions
- **Clean, minimal design** — Distraction-free writing experience
- **Responsive layout** — Works across desktop and mobile devices

## 🛠️ Tech Stack

- **Frontend Framework**: React
- **Language**: JavaScript / TypeScript
- **Styling**: CSS
- **State Management**: React Hooks (useState, useEffect)

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
   git clone https://github.com/ekienandaa/daily-journal-app.git
   cd daily-journal-app
```

2. Install dependencies
```bash
   npm install
```

3. Start the development server
```bash
   npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
